معرفی پروژه – MiniDigikala (نمونه فروشگاه اینترنتی)

این پروژه یک نسخه نمونه از فروشگاه اینترنتی شبیه به Digikala است که با هدف تمرین طراحی صفحات فروشگاهی حرفه‌ای و نمایش توانایی در UI/UX، تعاملات تعاملی و طراحی نزدیک به وب‌سایت واقعی توسعه یافته است.

ویژگی‌ها و پیاده‌سازی فنی

لودینگ صفحه با لوگوی Digikala
هنگام ورود به سایت، کاربران با یک صفحه لودینگ کوتاه مواجه می‌شوند که لوگوی Digikala را نمایش می‌دهد و پس از چند ثانیه وارد سایت می‌شوند.

نمای فروشگاهی شبیه Digikala
طراحی صفحه با تمرکز بر تجربه کاربری فروشگاهی و شبیه‌سازی ظاهر نسخه اصلی Digikala، شامل دسته‌بندی‌ها، منوها و زیرمنوها.

اسلایدرهای متنوع

اسلایدرهای تکی و گروهی با حرکات مختلف (دورانی یا خطی)

نمایش محصولات و پیشنهادات به شکل حرفه‌ای و جذاب برای کاربر

GIF و آیکون‌های حرفه‌ای
بخش GIF و آیکون‌ها در کنار هم طراحی شده‌اند تا تجربه دیداری زنده و حرفه‌ای ایجاد شود.

دسته‌بندی‌ها، منوها و زیرمنوهای تعاملی با JavaScript

همه منوها و زیرمنوها با JS تعاملی پیاده‌سازی شده‌اند

بخش پیشنهادها و DigiClub شامل زیرمنوهای قابل Hover و باز و بسته شدن هستند

منوها و زیرمنوها به صورت نرم و روان باز و بسته می‌شوند

Search Box و پشتیبانی تعاملی با JavaScript

بخش جستجو در گوشه پایین سمت راست به صورت انیمیشنی باز می‌شود

بخش پشتیبانی تعاملی نیز با JS پیاده‌سازی شده و امکان باز و بسته شدن آن فراهم است

تخفیف‌ها با تایمر تعاملی (JavaScript)

شمارشگر تخفیف با JS کار می‌کند و پس از اتمام زمان، پایان تخفیف را نشان می‌دهد

تعامل و نمایش زمان واقعی تخفیف به صورت پویا انجام می‌شود

تبلیغات لینک‌دار
تبلیغات با لینک مستقیم به صفحه مربوطه، امکان هدایت کاربر را فراهم می‌کنند

فوتر با دکمه Back-to-Top با JavaScript
دکمه برگشت به بالای صفحه با JS تعاملی پیاده‌سازی شده و کاربر را به آرامی به بالای صفحه هدایت می‌کند

دکمه ورود به حساب کاربری و برگشت به صفحه قبلی با JavaScript

دکمه “Go to Account” کاربران را به صفحه ورود هدایت می‌کند

دکمه Back با JS تعاملی کاربران را به بخش قبلی بازمی‌گرداند

ریسپانسیو حداقلی (نیمه ریسپانسیو)
طراحی اصلی برای دسکتاپ بهینه شده و نمایش در دستگاه‌های کوچک تا حدی پشتیبانی می‌شود، اما تمرکز روی تجربه دسکتاپ است.

طراحی با Bootstrap 4 و CSS خالص
Grid، کارت‌ها، منوها و بخش‌های مختلف با Bootstrap 4 طراحی شده‌اند تا ساختار استاندارد و حرفه‌ای حفظ شود.

جایگاه پروژه در مسیر رشد

این پروژه نشان‌دهنده توانایی من در طراحی فروشگاه اینترنتی حرفه‌ای با تعاملات پویا است و مهارت‌های زیر را به خوبی نمایش می‌دهد:

طراحی صفحات فروشگاهی با دسته‌بندی‌ها، منوها و زیرمنوهای تعاملی

پیاده‌سازی اسلایدرها و GIFهای جذاب

طراحی تایمر تخفیف، دکمه Back-to-Top و پنل پشتیبانی با JavaScript

تجربه Hover و انیمیشن‌های حرفه‌ای

طراحی دسکتاپ محور با ریسپانسیو حداقلی

جمع‌بندی

ایجاد منوها، دکمه‌ها، تایمرها و پنل‌های پشتیبانی با JavaScript
