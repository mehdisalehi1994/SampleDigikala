MiniDigikala:

یک نمونه فروشگاه اینترنتی الهام‌گرفته از Digikala است که با هدف شبیه‌سازی یک وب‌سایت فروشگاهی واقعی توسعه داده شده است. در این پروژه، بخش‌های مختلفی مانند دسته‌بندی‌ها، منوها و زیرمنوهای تعاملی، اسلایدرهای متنوع، تایمر تخفیف، جستجوی پویا، پنل پشتیبانی و دکمه‌های کنترلی با JavaScript پیاده‌سازی شده‌اند. ساختار صفحات با استفاده از Bootstrap 4 و CSS خالص طراحی شده و تمرکز اصلی پروژه بر نسخه دسکتاپ بوده است، با پشتیبانی محدود از نمایش در دستگاه‌های کوچک. این پروژه توانایی من را در پیاده‌سازی رابط‌های فروشگاهی تعاملی، مدیریت تعاملات کاربرمحور و ساخت ساختارهای استاندارد فرانت‌اند در سطحی نزدیک به پروژه‌های واقعی وب نشان می‌دهد.
